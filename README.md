## Introduction
nodedrafts is a full-stack blog application that allows users to create, edit, and delete blog posts. It also supports user authentication and comment functionality.
## Overview 📝
Nodedrafts is a feature-rich blogging platform that enables users to create and manage blog content with a modern, responsive interface. The application includes user authentication, real-time commenting, and a full CRUD interface for blog post management.

## Features
- User authentication (sign up, login, logout)
- Create, read, update, and delete blog posts
- Comment on blog posts
- Responsive design
## Key Features ✨
- **User Management**
  - Secure authentication (signup/login/logout)
  - JWT-based session handling
  - User profiles
- **Blog Features** 
  - Create, edit and delete posts
  - Rich text editing
  - Image uploads
  - Comment system
- **UI/UX**
  - Responsive design
  - Modern interface
  - Real-time updates

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/nodedrafts.git
    ```
2. Navigate to the project directory:
    ```bash
    cd nodedrafts
    ```
3. Install dependencies for both frontend and backend:
    ```bash
    npm install
    cd client
    npm install
    cd ..
    ```
4. Create a `.env` file in the root directory and add your environment variables:
    ```env
    MONGO_URI=your_mongodb_uri
    JWT_SECRET=your_jwt_secret
    ```
5. Create a `.env` file in the frontend directory (client folder) and add necessary frontend environment variables.
### Backend
- Node.js & Express
- MongoDB (database)
- JWT (authentication)

## Usage
1. Start the backend development server:
    ```bash
    npm run dev
    ```
2. In a separate terminal, start the frontend development server:
    ```bash
    cd client
    npm run dev
    ```
3. Open your browser and navigate to `http://localhost:3000`.
